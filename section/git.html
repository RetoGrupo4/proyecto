GIT Y GITHUB

-Creamos una cuenta en github y desde esa cuenta creamos un nuevo repositorio.

-Creado el repositorio nos saldra las posibles maneras de poder subir nuestros archivos al github

-Por la web desde github tambien es posible subir archivos

-Por el terminal mediante comandos estos son los pasos a seguir:

  • Instalar el git.
  • Vincular nuestro correo a git.
  • Iniciamos y creamos el repositorio dentro del directorio de trabajo.
  • Hacemos una copia
  • Consolidamos
  • Vinculamos a nuestro git (local) el nuevo directorio de trabajo (repositorio creado en la cuenta de GitHub)
  • Pasamos el proyecto de Git a GitHub, este comando solo se utilizara el origin master cuando lo usemos por primera vez. Nos pedira usuario y contraseña.
  • Para pasar a GitHub modificaciones usaremos solo el push. Pedira usuario y contraseña de nuevo
  • Una forma de subir al equipo algo que tengamos en el github seria utilizar el clone.
    • git clone HTTPS:\\--------------------
  • Nuestro repositorio quedaria de este modo despues de los comandos realizados para crearlo donde se mostrara tambien los commit realizados por las modificaciones que hagamos.
