LAMP

LAMP hace referencia a un grupo de diferentes programas de código abierto que típicamente son instalados en conjunto, con el objetivo de habilitar a un servidor como prestador de los servicios de páginas web dinámicas, así como los de aplicaciones web.

INSTALAR EL SERVIDOR WEB APACHE Y ACTUALIZAR EL CORTAFUEGOS USANDO LOS SIGUIENTES COMANDOS.

-Actualizar y descargar la lista de paquetes de los repositorios antes de instalar el apache.

  • sudo apt-get update
  • sudo apt-get install apache2

-Ajustar el cortafuegos para permitir el trafico de web.

  • sudo ufw app list

-Si solicitas la información del perfil Apache Full, se mostrara que el tráfico se encuentra habilitado para los puertos 80 y 443.

  • sudo ufw app info "Apache Full"

-Permitir el tráfico de entrada HTTP y HTTPS para este perfil.

  • sudo ufw allow in "Apache Full"

-Por ultimo comprobaremos con la direccion ip del servidor la pagina index.html por defecto.

MYSQL ES UN SISTEMA DE ADMINISTRACIÓN DE BASES DE DATOS.

-Instalar MySQL.

  • sudo apt-get install mysql-server

-Cuando la instalación esté completa, debes ejecutar un archivo de comandos de seguridad que viene preinstalado con MySQL.

  • sudo mysql_secure_installation

PHP PUEDE EJECUTAR ARCHIVOS, CONECTARSE A TUS BASES DE DATOS MYSQL PARA OBTENER INFORMACIÓN, Y MANEJAR LA VISUALIZACIÓN DEL CONTENIDO PROCESADO SOBRE TU SERVIDOR WEB.

-Instalar PHP.

  • sudo apt-get install php libapache2-mod-php php-mysql

-A continuación, reiniciamos el servidor Apache para que los cambios sean reconocidos, y se puede hacer mediante el comando:

  • sudo service apache2 restart
  • sudo service apache2 status
